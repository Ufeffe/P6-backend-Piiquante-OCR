# P6-backend-Piiquante-OCR

Steps to start the project:

1- Frontend :
 - clone repo from: https://github.com/OpenClassrooms-Student-Center/Web-Developer-P6.git
 - open a terminal 
 - cd frontend
 - npm init
 - npm start
 - open your browser on http://localhost:4200/


2- Backend :

 - open an other terminal
 - cd backend
 - npm init
 - nodemon server
